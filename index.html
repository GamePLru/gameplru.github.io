    Built-in monitoring of game servers and statistics collection system.

    GamePL has a built-in system for collecting statistics on consumption of ram, cpu and disk quota by game servers, as well as the maximum number of players online. All this is displayed on beautiful graphics that your customers can place on various sites. There is also a system of banners that players can use at their discretion. For CS, CSS, CSS v34, CS: GO servers, it is possible to sell admin / VIP through monitoring.

    Unlimited number of game servers, physical cars, tariffs and locations.

    You can connect an unlimited number of machines for game servers to GamePL, as well as create unique tariffs and locations for them. GamePL is able to control more than 10,000 game servers and at the same time, still work quickly.
